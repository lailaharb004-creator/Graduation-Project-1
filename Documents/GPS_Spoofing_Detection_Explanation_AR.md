# مفهوم Machine Learning

Machine Learning يشبه تعليم الطالب.

نعطي النموذج أمثلة:

    بيانات GPS 1 → Normal
    بيانات GPS 2 → Fake
    بيانات GPS 3 → Normal

بعد التدريب يستطيع تصنيف بيانات جديدة.

------------------------------------------------------------------------

# النماذج المستخدمة

## Random Forest

Random Forest عبارة عن مجموعة من Decision Trees.

كل شجرة تعطي توقع، وبعدها يتم اختيار القرار حسب الأغلبية.

مثال:

    Tree 1 → Fake
    Tree 2 → Fake
    Tree 3 → Normal
    Tree 4 → Fake

النتيجة:

    Fake

يستخدم عدد كبير من الأشجار لتحسين الدقة.

------------------------------------------------------------------------

## Neural Network (MLP)

شبكة عصبية تحاول تقليد طريقة عمل الدماغ.

تتعلم العلاقات المعقدة داخل البيانات.

التركيب:

    Input Layer
          ↓
    100 Neurons
          ↓
    50 Neurons
          ↓
    Output Layer

تستخدم Activation Function مثل ReLU لمساعدة النموذج على التعلم.

------------------------------------------------------------------------

## HistGradientBoosting

يتعلم بشكل تدريجي:

    Model 1 يعمل أخطاء
            ↓
    Model 2 يتعلم من الأخطاء
            ↓
    Model 3 يحسن النتيجة

وهذا يسمى Gradient Boosting.

# Feature Engineering

إنشاء Features جديدة تساعد النموذج على اكتشاف Spoofing.

## Satellite Ratio

القانون:

    sat_locks / sat_count

يقيس نسبة الأقمار المتصلة.

------------------------------------------------------------------------

## Satellite Discrepancy

القانون:

    |sat_count - sat_locks|

يقيس الفرق بين الأقمار المكتشفة والمتصلة.

------------------------------------------------------------------------

## Velocity Difference

يقيس تغير السرعة بين القراءات.

التغيرات الكبيرة قد تكون علامة على وجود مشكلة.

------------------------------------------------------------------------

## Heading Difference

يقارن بين اتجاه الحركة واتجاه GPS.

------------------------------------------------------------------------

## Rolling Statistics

ينظر إلى آخر عدة قراءات لفهم الاتجاهات وليس قراءة واحدة فقط.
